# NOIR — Premium Restaurant Template

A world-class multi-page fine dining website template.

## Features
- 6 pages: Home, Menu, The Kitchen, Gallery, Reservations, Contact
- Cinematic hero with parallax scroll
- Custom animated cursor
- GSAP ScrollTrigger scroll reveals + parallax
- Curtain page transitions
- Horizontal drag-scroll signature dishes section
- Sticky split-screen story section
- Masonry gallery with lightbox
- Tasting menu cards (2 options)
- Full reservation and contact forms
- Stats band with gold hover effects
- Team cards with bio reveal on hover
- Newsletter footer
- Fully mobile responsive

## Palette
Obsidian · Champagne · Dusty Rose · Off-White Cream

## Typography
Playfair Display (serif) + DM Sans (sans)

## Deployment
Pure static HTML — connect to Vercel and deploy instantly.
